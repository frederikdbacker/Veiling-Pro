# HORSE24-scraper geport naar main (Hannoveraner OnLive)

**Datum:** 26 juni 2026
**Thema:** De bestaande (nooit gemergede) HORSE24-scraperlogica naar main brengen
zodat Hannoveraner OnLive / Verdener Auktion Online-veilingen binnengehaald worden.
**Branch:** `feat/horse24-scraper`
**Schemawijziging:** geen.

## In gewone taal

De scraper-logica voor HORSE24 (Verden/Hannoveraner OnLive) bestond al op een oude
branch (`feat/collectie-url-import-multibron`, commit `ab1b6ba`) maar was nooit
naar main gemerged. HORSE24-sites zijn Vue-apps die hun data server-side als JSON
in de pagina meegeven — één gewone download geeft de hele collectie inclusief de
3-generatie-afstamming. Deze sessie is die bewezen logica geport en ingehaakt in
de huidige scraper-registry.

  lib **ongewijzigd** geport.

## Wat is gebouwd

1. **`scripts/lib/horse24.mjs`** — 1-op-1 geport uit `ab1b6ba` (parsing
   onaangeraakt). Exporteert `scrapeCollection(auctionUrl, opts)` → `{meta,
   horses}`; leest `:auction`/`:lots`/`:lot`/`:pedigrees` via pure fetch en
   bouwt de 3-generatie-afstamming in het bestaande `lots.pedigree`-formaat.
   De oude `scripts/lib/sources/*`-multibron-laag is bewust **niet** meegenomen.
2. **`scripts/scrape-horse24.mjs`** — standalone wrapper in de vorm van de andere
   `scrape-*.mjs`: leest `argv [auction-url, house-name, collection-name?]`, roept
   `scrapeCollection` aan en schrijft `data/horse24-<slug>.json` in het meta +
   horses[]-formaat dat `import-lots.mjs` inleest (collection/house/date/status/…).
   Harde fail bij 0 lots.
3. **Registry-entry `horse24`** (`src/lib/scraperRegistry.js`): `engine 'fetch'`,
   `script 'scrape-horse24.mjs'`, `importer 'import-lots.mjs'`, `match` op
   `verdener-auktion-online.com` én `*.horse24.com`, `houseHint` →
   'Hannoveraner Verband' voor de Verden/Hannoveraner-host (andere horse24-tenants
   → UI vraagt het huis), `needsHouseName: true`, `collectionName`-doorgifte
   (dedupe-op-link intact), `buildArgs` → `[rawUrl, houseName, collectionName?]`.
4. **Live canary** in `scripts/test-scrapers-live.mjs` (auction 137), met
   commentaar: wordt deze rood, check eerst of de pagina nog bestaat vóór je de
   scraper "stuk" noemt (een afgelopen veiling kan offline gaan) — net als bij
   pwb/livesauction.
